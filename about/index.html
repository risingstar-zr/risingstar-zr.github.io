项目经验

  • 朝夕绘馆学习平台(项目周期2017.03-2017.06)

  • 顾问通讯官网(项目周期2017.01-2017.03)

    “顾问通讯官网”是济南顾问通讯科技有限公司根据用户需求对所售卖产品“云控插排、路由器“等提供售前，售后的支持。
    项目职责：1、产品中心模块：包括产品分类查看，修改，发布；2、服务与支持模块：包括各类产品的资料上传，搜索、下载和批量删除；
    项目架构：LNMP+Thinkphp3.23+Ajax/JavaScript/Html5/Css3
    1、用MVC架构，LNMP生产环境；2、用SVN等版本控制工具；3、用PowerDesigner建模工具,Navicat数据库管理工具；

  • 江苏互联网安全监测通报平台(项目周期2016.04-2016.08)

    "江苏互联网安全监测通报平台”是江苏国瑞信安科技有限公司使用Java语言耗时五个月开发的一款应用于江苏省各地级市政府网站安全监测实时通报的信息系统，目前已经在江苏省公安厅上线使用。
    项目职责：1、基础数据模块：网站信息管理-网站修改，查询；2、系统管理模块：包括部门管理，角色管理，用户管理的查询、修改和批量删除;
    项目架构：SSM + Mysql + Power Designer + Svn + Maven

  • 高校e生活微信公众平台(项目周期2015.03-至今)

    高校e生活微信公众平台是面向济宁学院和德州学院全体在校学生的教务信息查询平台。目前用户量约10000人左右。
    主要功能：1、课表功能：查询课表，班级课表查询功能，课程搜索功能；2、成绩查询，自习室空座、图书信息等查询功能；3、空教室查询等功能；
    项目使用java web HttpClient 模拟登录教务系统，爬取相关信息，并整合成json格式数据API接口。然后利用ThinkPHP框架进行微信二次开发，项目部署在SAE并使用SVN进行团队协作开发
